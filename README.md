# Colour-Detection-Project

> ## 🚀 Project Overview
>
> 🎨 **Color Detection using OpenCV** is a Computer Vision project developed using **Python** and **OpenCV**.
>
> It allows users to click anywhere on an image to identify the detected color instantly. The system displays:
>
> - 🎨 Color Name
> - 🔢 RGB Values
> - ⚡ Fast Color Recognition
> - 🐍 Simple and Beginner-Friendly Code
>
> This project demonstrates the basics of **Image Processing**, **Computer Vision**, and **OpenCV**, making it an excellent project for students and beginners.
